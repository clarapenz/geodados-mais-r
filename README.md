## Geografia, Dados Públicos e R
#### Este é um repositório com o material didático do minicurso e da oficina apresentados no eixo temático "Geografia, Cartografias e Novas Tecnologias".

### Conteúdo

- **Minicurso** — Análises geográficas a partir de bases públicas de dados: fundamentos e fontes teóricas
- **Oficina** — Introdução ao uso da linguagem de programação R para tratamento de grandes bases de dados georreferenciadas
